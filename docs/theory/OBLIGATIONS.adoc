= OBLIGATIONS — the Equality Theory proof-obligation ledger
The single citable list. Cite `ET-n`, never a prose paraphrase.
:toc: left
:sectnums!:

Rules of the ledger:

* Every obligation has a stable ID (`ET-1`..`ET-15`), a one-sentence
  statement (the full statement lives in the linked theory doc), a prose
  source, a status per the legend in link:README.adoc[docs/theory/README],
  and — where one exists — a mechanization slot.
* `docs/status/PROOF-STATUS.adoc` mirrors the status column and adds the
  prover evidence; if the two files disagree, PROOF-STATUS's CI drift gate
  is the alarm and this file is the authority on *statements*.
* Adding, removing, or renumbering an obligation is a theory change:
  ADR required.
