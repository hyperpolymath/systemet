= systemet — Theory
The formal home of Equality Theory: one document per layer and gate, plus the citable obligation ledger.
:toc: left
:sectnums:

This directory is the formal counterpart of the prose in the repository root
(`README.adoc`, `EXPLAINME.adoc`). The root files are the public orientation;
these documents are the specification. Where the two disagree, that is a bug —
open an issue.

== Status legend

Every document and every obligation carries one of these statuses. The words
mean exactly this and nothing more:

[cols="1,4",options="header"]
|===
| Status | Meaning

| `specified`
| The statement is written down precisely enough that a proof attempt (or a
  counterexample hunt) can begin. Nothing is proven by this status.

| `sketched`
| The discipline is described but its rules are not yet stated precisely
  enough to attempt a proof. Below `specified`.

| `mechanized`
| A machine-checked development discharges the statement, the prover actually
  ran (locally and in CI), and the artefact is listed in
  `docs/status/PROOF-STATUS.adoc` with its file path. `#print axioms` (or the
  prover's equivalent) confirms no stubbed dependency.

| `OPEN`
| The statement is a target, not a result. It must never be cited as proven,
  in this repository or downstream.
|===

== Obligation numbering

Proof obligations are numbered `ET-1` .. `ET-15` and live in
link:OBLIGATIONS.adoc[OBLIGATIONS.adoc]. That file is the only citable ledger:
`docs/status/PROOF-STATUS.adoc`, downstream issues (anytype, AffineScript), and
any paper cite obligations by `ET-n`, never by prose paraphrase. Adding,
removing, or renumbering an obligation is a theory change and requires an ADR
(see EXPLAINME "Relationship to anytype and AffineScript").

[source,text]
----
systemet    (this repo)   the theory — semantic authority
   |
   v  pins, implements, may extend but not redefine
anytype                    the reference kernel
   |
   v  product profile: grade = affine + cost

If implementation pressure in anytype or AffineScript forces a theory change,
it lands here as an ADR first — never as a quiet downstream edit.
